# 就Tensor.h的文档说明

# private部分

    首先，包括基本的数据区，形状信息，步长信息；  
    其次，为了让后续部分能正常运行，统一步长的移动、访问、计算等，to_flat_idx实现多维索引映射为一维坐标，count_elements部分主要是检测后续在reshape等形状变化中前后是否一致，保证操作安全。

# public部分
    
    主要针对题目要求实现功能，在此基础，主要增加  
    1.偏置加法：在执行矩阵加法后，加入bias_add完成偏移修正，更好的拟合复杂函数；  
    2.重载加号运算符：让两个张量直接使用"+"，让加法更直接；  
    3.广播broadcast:自动扩展张量形状，让不同的形状满足兼容条件，从而进行逐元素操作；  
    4.he_normal_init:给权重参数初始化，具有随机性，保证多样性； 
    5.zero_init：给偏置参数初始化，保证模型初期稳定性。
