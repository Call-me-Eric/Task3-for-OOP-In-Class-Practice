# 需要完成的内容

对于函数接口

## classExample

```cpp
int classExample(string str)
```

传入一个 $string$ 类型的字符串，需要提取出其中的整数，并返回它们的和，以 $int$ 类型返回。

## extractedQuote

```cpp
 extractedQuote()
```

需要自行输入一行（按照PDF的要求，分别使用>>,getline()实现），并按照PDF的要求输出结果。

## FileStream

```cpp
void FileStream(string FileName)
```

需要按照PDF要求实现，不过操作文件由FileName传入

## countWords

```cpp
int countWords(string Filename,string pattern)
```

传入两个参数 $Filename,pattern$，要求打开名字为 $Filename$ 的文件，并且返回其中 $pattern$ 单词出现的次数。

如果测试文件没有自行构造（实在不是我懒）
